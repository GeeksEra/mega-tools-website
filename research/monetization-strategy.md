## STREAM 1: DISPLAY ADVERTISING

### Ad Networks (Ranked by RPM)
1. **Ezoic** — Best for 10K-500K monthly visitors. RPM: $4-12
2. **Mediavine** — Requires 50K sessions/mo. RPM: $8-25. Apply at Month 4-5
3. **AdThrive/Raptive** — Requires 100K pageviews/mo. RPM: $10-30. Apply at Month 6
4. **Google AdSense** — Use to start, RPM $2-5, apply on launch

### Ad Placement Strategy (Maximize Revenue Without Hurting UX)
- Banner ads: 728x90 leaderboard above tool area
- Sidebar: 300x250 rectangle (desktop only)
- Below tool output: 300x250 or responsive
- Between tool results and FAQs section
- DO NOT place ads inside the tool interface (kills conversion)
- MAX 3 ads per page (Google's policy + UX)

## STREAM 2: FREEMIUM PRO PLAN

### Free vs Pro Features

| Feature | Free | Pro ($9.99/mo) | Business ($29.99/mo) |
|---------|------|----------------|---------------------|
| Daily tool uses | 5/day | Unlimited | Unlimited |
| File size limits | 5MB | 100MB | 500MB |
| Batch processing | No | Yes (10 files) | Yes (100 files) |
| Ads shown | Yes | No ads | No ads |
| Download formats | 1 format | All formats | All formats |
| API access | No | No | 10K calls/mo |
| Priority processing | No | Yes | Yes |
| History/saved files | 7 days | 30 days | Unlimited |
| Support | Community | Email | Priority |

### Pricing Strategy
- Pro: $9.99/month OR $79/year (save 34% — drives annual subscriptions)
- Business: $29.99/month OR $239/year
- Team: $49.99/month (up to 5 users)
- Enterprise: Custom pricing (contact us)

### Conversion Optimization
- Show "You've used 4/5 free uses today" counter (creates urgency)
- Pro upgrade prompt after 5th use (don't block, just prompt)
- "Try Pro free for 7 days" — free trial reduces friction
- Stripe payment processing (most trusted, global)
- Annual plan shown first (higher LTV)
- Show social proof: "Join 50,000+ pro users"

### Target Conversion Rates
- Free to Pro conversion: 1.5-2.5% (industry standard for tools)
- At 100K monthly visitors × 2% = 2,000 pro users
- 2,000 × $9.99/mo = $19,980/month from pro alone
